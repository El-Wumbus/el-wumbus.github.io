el-wumbus.github.io
===================

This is my personal website built on the vue framework using Typescript along with css and bootstrap css for styling.

Build Dependencies: 

* nodejs
* npm

.. code-block:: bash

  git clone https://github.com/el-wumbus/el-wumbus.github.io
  cd el-wumbus.github.io
  npm install
  npm run lint 
  npm run build

  # the built webpage is in the dist/ directory
